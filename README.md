- `Gurobi` (https://a-farahat-15071.readthedocs.io/en/latest/installing_gurobi.html)
- `jupyter notebook`
- `numpy`
- `pandas`
- `matplotlib`

Go to the `notebook_seating_project_Air_France.ipnyb` notebook.
